# Image Generation Style Guide

为本项目生成配图时，统一使用 A4 竖版中文技术教材信息图风格。

## Global Style

- 画幅使用 A4 纸竖版比例。
- 背景必须是纯白色。
- 风格是清爽、克制、现代中文技术教材信息图。
- 使用蓝绿色作为主色，暖黄色作为强调色，深灰色作为文字颜色。
- 使用模块化卡片、简洁箭头、节点、流程线、少量抽象数据元素表达概念。
- 图片可以包含中文文字，但文字必须清晰、短句化、标签化、模块化。
- 版式不固定，应根据页面内容自由选择最适合的图解方式。

## Text Rules

- 可以使用中文标题、短标签和短说明。
- 每个文字块尽量短，不写长段说明。
- 复杂含义优先用图形关系表达，文字只做辅助标签。
- 避免密集小字、公式墙、长句堆叠。

## Avoid

- 不使用渐变背景、深色背景、纹理背景、照片背景。
- 不使用赛博朋克、霓虹、复杂 3D、黑色高对比背景。
- 不使用拟人化机器人作为主角。
- 不使用真实品牌标识。
- 不让装饰元素抢过教学内容。

## Reusable Prompt

A4 竖版中文技术教材信息图，纯白背景，清爽克制的现代教育设计。使用蓝绿色主色、暖黄色强调色、深灰中文文字。画面由清晰的模块化卡片、简洁箭头、节点、流程线和少量抽象数据元素组成。中文文字清晰可读，短句化、标签化、模块化。每张图只解释一个核心概念，适合初学者理解。避免渐变背景、深色背景、照片质感、赛博朋克、复杂 3D、拟人化机器人、密集小字和长段文字。
