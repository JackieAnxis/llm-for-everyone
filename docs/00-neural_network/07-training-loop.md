## 训练循环是什么

前面几章里，我们已经分别看过前向传播、损失函数、反向传播和参数更新。训练循环，就是把这些步骤按顺序连起来，并且反复执行。

模型先用当前的权重和偏置做前向传播，得到预测；再用损失函数比较预测和真实答案，知道这次错得多不多；接着用反向传播把错误信号传回各层；最后根据这些信号进行参数更新。更新之后，模型又回到开头，再做下一次预测。

这就是训练最核心的闭环：

```text
预测 -> 计算错误 -> 调整参数 -> 再预测
```

训练不是只预测一次，也不是知道错了就停下。它更像反复练习：先作答，再批改，再订正。

## 一轮训练包含哪些步骤

一轮训练可以理解成一次“预测和订正”。继续用小测预测的例子，输入可能是：

```text
学习时长 = 2
复习次数 = 1
```

第一步是前向传播。模型把输入一层层向后传，最后给出预测：

```text
预测通过概率 = 0.3
```

第二步是计算损失。假设真实答案是通过，也就是目标更接近 `1`，那么 `0.3` 就明显偏低。损失函数会把这个差距变成一个数字，表示这次错误有多严重。

第三步是反向传播。模型不能只知道“错了”，还要知道哪些权重和偏置更该调整。反向传播会从输出层开始，把错误信号往前传，让各层参数得到调整线索。

第四步是参数更新。训练过程根据这些线索，小幅修改相关的权重和偏置。更新后，模型已经不再和刚才完全一样。

一轮训练可以简写成：

```text
输入样本
-> 前向传播得到预测
-> 损失函数计算错误程度
-> 反向传播传回错误信号
-> 参数更新
```

## 为什么要训练很多轮

参数通常不是一次就能调好的。刚开始时，模型内部的数字还不合适，可能把该通过的学生预测成不通过，也可能对某些样本只是碰巧猜对。

如果只训练一轮，模型得到的反馈太少，调整也太少。就像做题只订正一次，很难马上掌握一类题。模型需要反复经历“预测、计算错误、调整参数、再预测”，才能形成更稳定的判断。

多轮训练之所以可能逐步降低损失，是因为每次更新都会基于当前预测和损失调整参数，后续预测会在更新后的参数上继续进行。新的预测会带来新的损失，新的损失又会带来新的调整线索。只要整体方向有帮助，模型在许多样本上的预测就可能越来越接近真实答案，整体损失也就有机会下降。

当然，损失不一定每一轮都严格变小。训练看的是长期趋势，而不是要求每一步都完美。

## 参数更新后预测会变化

参数更新后，同一个输入也可能得到不同预测。原因很直接：前向传播依赖权重和偏置；这些数字变了，输出分数也可能变。

例如同一个学生，更新前可能是：

```text
学习时长 = 2
复习次数 = 1
预测通过概率 = 0.3
```

更新后再输入同样的信息，结果可能变成：

```text
预测通过概率 = 0.45
```

这不代表模型已经完全正确，只说明参数变化会影响之后的预测。

## 把完整流程串起来

把前几章合在一起，可以得到完整训练流程：

```text
准备样本和真实答案
-> 使用当前参数进行前向传播
-> 得到预测
-> 用损失函数比较预测和真实答案
-> 得到损失，知道错误程度
-> 通过反向传播把错误信号传回各层
-> 根据错误信号进行参数更新
-> 进入下一轮
```

前向传播回答“现在预测什么”，损失函数回答“错得多不多”，反向传播回答“错误信号怎样传回参数”，参数更新则真正改变模型内部的数字。

可以把这些步骤放在同一个循环里理解。很多次小调整累积起来，模型才有机会逐渐变好。

## 小结

训练循环把前向传播、损失函数、反向传播和参数更新连成一个整体。它的核心是：

```text
预测 -> 计算错误 -> 调整参数 -> 再预测
```

训练需要很多轮，因为参数通常不能一次调好。每一轮都会根据当前预测和损失提供新的反馈，参数也会随之改变。参数更新后，即使输入相同，下一次前向传播也可能得到不同预测。模型就是在这样的反复循环中，逐渐学会更合适的判断方式。
